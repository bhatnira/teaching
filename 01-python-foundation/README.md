# Plan 1: Python Foundation

**AI-Driven Development Training**  
**February 2026**

## Overview

This module covers the fundamentals of Python programming for entry-level learners. The course is designed to be completed in approximately 2-3 weeks with hands-on labs and assessments.

## Course Contents

### Lectures (13 Lessons)
- **Lesson 1-2**: Introduction, Variables, Data Types
- **Lesson 3-4**: Operators, Input/Output
- **Lesson 5-6**: Strings, String Methods
- **Lesson 6A-6B**: Conditionals (if/elif/else), Error Handling
- **Lesson 7-8**: Lists, Tuples
- **Lesson 9-10**: Sets, Dictionaries
- **Lesson 11-12**: List Comprehensions, Lambda Functions
- **Lesson 11B**: NumPy & Pandas Data Structures
- **Lesson 13**: File I/O, JSON, Regex, Modules

### Files

| File | Description |
|------|-------------|
| `lession-lab-1-AIDD-Training.tex` | Main lecture slides (LaTeX Beamer) |
| `lab-solutions.tex` | Solutions for all lab exercises |
| `homework-assignment.tex` | 20 homework problems (100 points) |
| `homework-solutions.py` | Complete homework solutions with explanations |
| `quiz-questions-answers.tex` | 50 quiz questions with answers (LaTeX) |
| `quiz-answer-key.py` | Quiz answer key with rationale |

## Target Audience

- Entry-level programmers
- No prior Python experience required
- Basic computer literacy assumed

## Topics Covered

### Part 1: Python Basics
- Variables and data types
- Operators (arithmetic, comparison, logical)
- Input/output operations
- String manipulation

### Part 2: Control Flow
- Conditional statements (if/elif/else)
- Loops (for, while)
- Error handling (try/except)

### Part 3: Data Structures
- Lists, tuples, sets, dictionaries
- List comprehensions
- Lambda functions

### Part 4: File Handling & Data
- File I/O operations
- JSON processing
- Regular expressions
- Introduction to NumPy & Pandas

## Assessment Structure

| Assessment | Points | Questions/Problems |
|------------|--------|-------------------|
| Lab Exercises | - | 13 labs (2 scenarios each) |
| Homework | 100 | 20 problems |
| Quiz | 100 | 50 multiple choice |

## How to Use

### For Instructors
1. Compile LaTeX files to PDF for presentations
2. Use lab exercises during class
3. Assign homework for independent practice
4. Use quiz for assessment

### For Students
1. Follow lecture slides
2. Complete lab exercises in class
3. Submit homework within 1 week
4. Review quiz answers for self-study
